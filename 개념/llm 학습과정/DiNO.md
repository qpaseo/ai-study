DiNO :

- 자가지도 학습의 일부로 라벨이 붙어있지 않은 데이터(이미지)도 학습할수 있다는 방법
  - 많은 이미지를 학습에 사용가능
- 이를 통해 pretrain와 비슷하게 대규모 데이터를 통해 얻는 이점인 보는것에 대한 이해를 적용할수 있음
- 어느정도의 pretrain를 해결할수 있어보임

<br><br>
DiNO가 만든 feature map :

- 라벨이나 특정 지침없이 2d이미지 기준으로 사진속에 있는 사물을 분리할수 있다고 함
  - 마치 정답 라벨을 넣을때처럼 사물을 구분할수 있더라

<br><br>
DiNO학습 방식 :

- 기본 : 라벨과 같이 이미지를 가지고 학습하는게 아니라 이미지만 가지고 학습하는 방식
  - 활용방법 : 이렇게 이미지만 가지고 학습하는 이미지 모델이 가능하면 파인튜닝 학습에 도움이 됨
    - 특정일을하게 학습 시키는 파인튜닝은 답을 주고 학습하는게 도움이 됨
    - => 최종적으로 거의 모든 이미지를 파인튜닝 학습에 사용할수 있음 (라벨링이 많이 필요없이)

- 상세 :
  - DiNO는 이미지의 라벨을 달지않고 학습할수 있게 하는 방법인데, 이 방법에서 무언가 라고 예상한거를 매칭시키는 방법
  - Distillation
    - 학습을 하기는 하는데 교수의 출력을 정답이라고 가정하고 같아지도록 학습
      - 교수 : 출력이 같게 나오게 설정되어있는 다른 딥러닝 모델
      - 여기서 교수는 teacher, 따라가는 모델을 student 라고 함
    - 한계 
      - teacher 모델 이상으로 똑똑해질수는 없음
  - EMA(Exponential Moving Average)
    - 모델 학습중 가중치가 튀어서 모델 학습에 방해되는 값이 모델에 많은 영향을 받지 않게 가중치를 평균으로 잡아주는것
      - 가중치 안정화
      - 평균적으로 EMA를 적용한 모델이 좀더 성능이 좋다고 함
        - => 안정적인 teacher 모델에 사용(EMA를 적용해서 성능이 좋아진다고 가정하면 자신보다 항상 좋은 모델로부터 전수 받을수 있다고 생각하는거임)
    - Collapse(붕괴)
      - 모델이 데이터의 특징을 학습하고 그 학습에 따라 정답률을 따라 가는것이 아니라 정답과의 차이만 인위적으로 낮추는 현상 
      - DiNO에서의 Collapse
        - 만일 student가 잘못 학습하기 시작해서 teacher도 영향을 받아 최종적으로 teacher가 출력이 동일하도록 고정이 되어버린다면 student가 그대로 따라가기 떄문에 학습이 의미가 없어져 버린다는것
    - DiNO가 Collapse을 피하는방법
      -  Centering
         -  이유 : 만일 어떤 이유로 student모델이 같은 값만 반환하도록 해서 teacher도 영향을 받아 
            teacher도 같은 값만 반환된다면 => 학습이 의미가 없음
         -  해결 방법 : 자기 자신의 평균을 빼는것 => 특정 출력으로 쏠리는걸 방지할수 있음
      -  Sharpening
         - 이유 : 만일 어떤 이유로 student모델의 출력 확률이 1을 나눠가진다면(하나가 조금 차이로 높다고 한다면), 
         teacher도 영향을 받아 teacher도 같은 값만 반환된다면 => 학습이 의미가 없음
         - 해결 방법 : 값을 0.04로 나누어서 각 logit(클래스 분리에 해당하는 가능성[*소프트맥스 적용 전 버전])의 차이를 키우는 방법
           - 소프트맥스? : logit은 합이 1이 아닐수도 있는 원시적인 값이기 떄문에 이를 정규화 해주는 함수 
             - 정규화 방법
               - 1. 로짓 사이의 상대적인 차이를 지수적 상승을 사용해서 벌림
               - 2. 변형된 값들의 e를 취해서 전채 합으로 나눔 => 합이 1이됨
               - +. 이러면 결과 값들의 차이가 극명하게 벌어지게 됨
      - Centering & Sharpening 적용하면?
        - Centering으로 한쪽으로 치우치는것을 방지
        - Sharpening으로 값이 평평해지는걸을 방지
        - => 같은 종류 입력만 같은 출력을 내도록 학습기능
    - multi-crop 
      - 이미지를 여러 뷰로 학습하는거 (ViT보고 오면 판함)
      - 이미지 일부만 랜덤으로(학습을 위해 [크기만 랜덤이고 자르는 조각{path}의 크기는 같음]) 입력하는 기술
        [ViT](./ViT.md)
      - global view
        - 이미지중 local view보다 큰 부분을 crop(자른거)한거
        - teacher, student 모델에 사용됨
      - local view
        - 이미지중 global view보다 작은 부분은 crop(자른거)한거
        - student 모델에 사용됨
      - view 적용방식
        - student는 local,global view를 받으면서 teacher의 최종 답변을 맞추면서 학습함
        - 각 path(정보)의 상관관계를 학습할수 있음
