자가지도 학습 (Self-Supervised Learning) <br> - 일부러 빈칸, 빈 부분을 생성한 뒤에 원래대로 맞추는 가상의 문제를 해결하며 학습하는 방법<br> - data와 답인 label을 주는것이 아니라 data를 보고 (또는 일부만 보고) 패턴을 학습하는 방법<br> - 데이터를 이루는 규칙(semantic)을 학습함

1 pretrain(프리 트래이닝) 사전학습 : 방대한 양의 사전 정보를 가지고 이 새상의 언어들의 "규칙"을 학습하는것<br> - 어떤 문자뒤에 뭐가 와야 하는지<br> - 어떤 문자 엎에는 뭐가 안오는지<br> - 단어 나열의 패턴을 학습<br> - 자가지도 학습의 일부<br> - 학습하고자 하는 언어 전까지의 데이터를 보고 바로 다음 단어를 예측하는 방식으로 학습<br> - 자연어 이기만 하면 학습에 사용 가능<br>

2 finetuning 파인튜닝 : 어떤 말을 해야할지 설정하는 학습<br> - 데이터와 데이터와 매칭되는 어떤 값을 통해 도덕성, 진위여부, 논리적 의미 등을 학습<br> - pretrain 보다는 적은 데이터 셋으로 학습함<br> - 라벨이나, 데이터에 대응되는 값이 있어야 한다<br> - 지식이나 할말이나 안할말을 구분하는 단계라고 할수 있음<br>

한계<br> - 1차원 데이터 나열 개념이 아니면 적용하기 힘들다<br> - 언어는 전까지의 단어를 가지고 와서 다음단어를 예측 하는 건데, 이미지는 2d여서 다음을 예측 하는게 어렵다<br> - 이미지는 언어에 비해 redundancy(중복)이 많다<br> - 언어는 언어와 언어사이에 연결이 잘 되지만 이미지는 이미지와 다음? 이미지 사이에 중복되는 부분이 많다.<br>

이미지 모델<br> - pretrain이 잘 맞지 않아서 이걸 안하면 시각적 정보에 대한 일반적인 이해가를 할수 없음<br> - 다양한 비전 task (일)을 할수 없다, (특정한 일만 잘하고 다른 일(특정한)은 잘 못함)<br> - 그러니 여러가지 작업을 한번에 하지 못함 (이미지 깨짐)<br> - 이런 한계를 해결하기 위해서 나온 방법이 DiNO<br>
