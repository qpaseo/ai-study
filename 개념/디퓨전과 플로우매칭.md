개요
1. 단순 text생성이 아니라 이미지나 비디오 생성에서 사용되는 학습 방법
2. 아래 2 방법은 이미지에 노이즈를 넣으면 원본 이미지를 찾아가는 방법으로 학습 함

디퓨전
1. 이미지에 노이즈를 넣으면 모델이 어떤 노이즈를 제거해야 하는지 예측해서 지우도록 하는 방법
2. 조금식 "랜덤하게" 걷어내며 원본 이미지(데이터)를 생성

플로우 매칭
1. 노이즈를 제거하고 원본 이미지로 가는 백터를 직접 바로 계산하는것
2. 바로 원본 백터로 가서 원본 이미지(데이터)를 생성

차이
1. 디퓨전은 랜덤이 포함되므로 "호들갑스럽게"돌아갈수 있는 경향이 있음
2. 플로우 매칭이 "비교적" 효율적으로 이미지(데이터)를 생성 가능

디퓨전 사용 이유
1. 랜덤성을 이용해서 이미지를 다양하게 만들수 있도록 학습할수 있음
2. 이미 많은 연구가 된 방법이여서 많은 기술이 디퓨전으로 최적화 되어있는 경우가 있음
3. 플로우 매칭은 백터를 직접 바로 연산하는 것이기 때문에 분포가 복잡하면 이 분포를 계산하게 하는것 이 학습 과정이 더 힘들수도 있음

결합
1. 디퓨전의 탄탄한 기본기에 플로우 매칭의 효율적인 샘플링 기법을 결합하는 방법도 연구중이라고 함

역전파
1. text와 비슷한 방식으로 역전파를 진행한다고 함, 
2. 단 이미지 에서의 역전파의 문제 정의가 약간 다른데 text는 단어를 맞추었는지, 이미지는 "모델이 예측한 노이즈와 실제 노이즈 사이의 오차" 를 주로 사용한다고 함
