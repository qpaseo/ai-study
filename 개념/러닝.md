개요
1. 러닝 : 컴퓨터가 데이터를 바탕으로 스스로 학습하고 패턴을 파악하여 사람처럼 사고하거나 작업을 수정해도록 학습 시키는 것


머신러닝
1. 개요 : 규칙을 코딩하지 않고, 데이터를 주고 학습시켜서 컴퓨터가 스스로 규칙을 찾아내게 만드는 기술
2. 사용이유 : 
   1. 규칙을 하나하나 만들수 없기 떄문에
   2. 미래를 예측하기 위해
   3. 자동화를 위해
3. 사용 예시
   1. 자율주행
   2. 게임 활용
4. 특징
   1. 정형화 되어있는 데이터에 강함
   2. 데이터 양이 상대적으로 적을때 강함
   3. 단순한 결과가 아닌 설명이 필요할때 강함


딥러닝
1. 개요 : 특징을 찾아내기 위한 기능(신경망)이 많이 쌓여 있다는 것, 복잡한 문제를 풀수 있도록 설계하는것
2. 인공 신경망 : 노드들이 층으로 이루어 연결된 구조
3. 심층 구조 : 단순히 입력, 출력으로 나누어진게 아니라 그 사이에 데이터를 처리하는 은닉층을 넣어 데이터를 스스로 추출하게 하는 구조
4. 특징 자동 추출 : 사람이 특징을 알려주지 않아도 스스로 데이터 패턴을 파악할수 있음
5. 사용 예시 : 
   1. 정형화 되어있지 않은 데이터, 
   2. 양이 많은 데이터, 
   3. 문제가 복잡한 데이터(단순 코딩이나 구조로 해결하기 어려운 것)
6. 좋은점 : 
   1. 사람이 프롬프팅을 하지 않고 ai가 스스로 학습하여 결과를 도출가능,
   2. ai가 스스로 학습 가능
   3. 어느정도 학습하면 높은 정확도
   4. 하나의 모델로 여러 작업 가능 (학습은 따로 해줘야 함)
   

구조
1. ai -> 머신러닝 -> 딥러닝


상세 구분
0. 개념
   1. Encoder : 특징을 찾아내는거
   2. Feature : 데이터 압축해서 만든 핵심정보(숫자로 변경해서 특정한 특징을 저장해 놓는것 [이거 가지고 Decoder에서 도출])
   3.Decoder : 정답을 도출하는것
1. 지도학습 : 문제와 정답을 알려주는것 
   1. 스스로 유추해서 답을 내는 방법을 학습
   2. 회귀문제 : 값을 내는 방식 ex) 환율 
2. 비 지도학습 : 답을 주지 않고 문제만 주는것
   1. 목적 : 군집화(유사도를 기반으로 군집으로 묶는것)나 차원 축소(중요한 정보만 남겨두는것)
3. 준 지도학습 : 문제에 대한 답을 줄때도 아닐때도 같이 학습 하는것
4. 강화학습 : 문제를 주고 답이 나오면 점수에 따라 피드백을 진행해서 목표치까지 반복하는것
