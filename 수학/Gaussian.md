**Gaussian** : 정규 분포중 대칭을 이루는 종모양의 분포
    - 형균 : 분포의 정중앙
    - 표준편차 : 곡선의 너비를 결정

**그래서 ai에서 많이 이용되는 이유가?**
1. 현실 데이터와 그 속에서의 노이즈가 가우시안 분포를 따라서
2. 수학적으로 계산하기가 편해서 => 다루기 쉽다
3. 생성형 ai중 많은곳에서 사용해서 => 연구가 많이되어서
4. 모델 가중치 초기화 => 맨 처음 가중치를 설정할때에 0대신 랜덤으로 설정하는데 이 때의 값을 가우시안 분포에서 무작위로 추출한 값으로 사용한다.
5. 가우시안 프로세스 및 베이지안 최적화
   1. 가우시안 프로세스
      1. 중심선(구분한 것의 값)과의 차이를 두고 차이가 크면 클수록 데이터를 잘 모른다(데이터가 없다)를 수학적으로 모델이 계산할수 있게 해서 신뢰도를 계산할수 있게 해주는 방법
   2. 베이지안 최적화
      1. 사람이 설정해줘야 했던 값 (레이어 수, 배치, 크기 등[하이퍼파라미터])을 용도에 맞게 조정해나가야 성능이 좋아지는데 이 값을 사람이 조절하기에는 값이 너무 오래걸려서 
      이를 ai가 선택해 가면서 조절할수 있게 해주는 방법
      순서
        - 현제 파라미터 기반으로 가우시안 프로세스를 사용해 "지도"(시도해봐서 데이터가 높고 확실한곳 아닌곳을 표시한 지도)를 그림
        - 수학적으로 계산후 지도중 파라미터를 학습할 곳으로 이동
        - 시도후 "지도"에 반영
